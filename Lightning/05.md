# Creating a Library Package

```
MyLib
├── Makefile
├── Project.toml
├── README.md
├── docs
│   ├── Project.toml
│   ├── make.jl
│   └── src
│       └── index.md
├── build
│   ├── Project.toml
│   ├── build.jl
│   ├── extra_precompile.jl
│   └── generate_precompile.jl
├── install.sh
├── src
│   ├── MyLib.jl
│   └── mylib.h
├── test
│   ├── Project.toml
│   └── runtests.jl
└── uninstall.sh -> install.sh
```
